# 外壳

外壳结构也区分很多种，不过我接触的太少，不是很了解，就不班门弄斧了。

有的是单独底壳，直接安装电路板，安装轴体键帽的，也有区分上下外壳的。

从材质上来讲，大部分都是塑料，中高端的有铝坨坨。塑料不是低端的区分，尤其是透明键盘，普遍价位比较高端。

外壳一般和主板 PCB、定位板强相关。如选了某个 68 配列的外壳，其 PCB 也被固定在某些特定型号上，反之亦然。
